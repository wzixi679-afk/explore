# 📚 日常用语

## 问候和告别 (Greetings and Farewells)

| 英文 | 中文 | 例句 |
|------|------|------|
| Hello | 你好 | Hello, how are you? |
| Hi | 嗨 | Hi, nice to meet you! |
| Good morning | 早上好 | Good morning! |
| Good afternoon | 下午好 | Good afternoon! |
| Good evening | 晚上好 | Good evening! |
| Goodbye | 再见 | Goodbye, see you later! |
| See you | 再见 | See you tomorrow! |
| Bye | 拜拜 | Bye! |
| Night (Good night) | 晚安 | Good night, sleep well! |
| Take care | 保重 | Take care of yourself! |

## 基本礼貌用语 (Basic Polite Expressions)

| 英文 | 中文 | 例句 |
|------|------|------|
| Please | 请 | Please sit down. |
| Thank you | 谢谢 | Thank you very much! |
| Thanks | 谢谢 | Thanks for helping me. |
| You're welcome | 不客气 | You're welcome! |
| Excuse me | 对不起（打扰） | Excuse me, can I ask a question? |
| Sorry | 对不起 | Sorry, I'm late. |
| No problem | 没问题 | It's broken. No problem, I'll fix it. |
| It's okay | 没关系 | It's okay, don't worry. |
| Pardon me | 请再说一遍 | Pardon me? |
| My apologies | 我道歉 | My apologies for the mistake. |

## 自我介绍 (Self Introduction)

| 英文 | 中文 | 例句 |
|------|------|------|
| My name is... | 我的名字是... | My name is Tom. |
| I'm... | 我是... | I'm Wang Zixi. |
| Nice to meet you | 很高兴认识你 | Nice to meet you! |
| How do you do | 你好（正式） | How do you do? |
| Where are you from? | 你来自哪里？ | Where are you from? |
| I'm from... | 我来自... | I'm from China. |
| What do you do? | 你做什么工作？ | What do you do? |
| I'm a... | 我是一个... | I'm a student. |
| How old are you? | 你多大了？ | How old are you? |
| I'm ... years old | 我...岁 | I'm 25 years old. |

## 日常问候对话 (Daily Conversation)

| 英文 | 中文 |
|------|------|
| How are you? | 你好吗？ |
| Fine, thank you. And you? | 很好，谢谢。你呢？ |
| I'm doing well, thank you. | 我很好，谢谢。 |
| Not bad. | 还不错。 |
| How's everything? | 一切都好吗？ |
| What's up? | 怎么样？ |
| Not much. | 没什么。 |
| What's new? | 有什么新消息吗？ |
| Same old, same old. | 老样子。 |

## 常用回应 (Common Responses)

| 情况 | 英文 | 中文 |
|------|------|------|
| 同意 | Yes | 是 |
| 不同意 | No | 不 |
| 确认 | OK / Okay | 好的 |
| 不确定 | Maybe | 也许 |
| 不知道 | I don't know | 我不知道 |
| 理解 | I understand | 我理解了 |
| 不理解 | I don't understand | 我不理解 |
| 重复 | Can you say that again? | 你能再说一遍吗？ |
