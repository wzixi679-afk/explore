# 📅 每日学习计划详细指南

## 🎯 计划概述

这份计划为你提供每天1小时英语学习的**详细时间表和具体任务**。

**每天总时间：60分钟**

---

## 📋 标准每日计划（1小时）

### 🌅 早上计划（推荐 7:00-8:00 AM）

#### 09:00-09:15 | 词汇热身 (15分钟)

**目标**：唤醒大脑，开始英语学习

**具体任务**：
1. 打开 Quizlet 或 Anki
2. 复习昨天学的10个单词
3. 预览今天要学的5个新单词
4. 大声读出来3遍

**推荐资源**：
- Quizlet: www.quizlet.com
- Anki: www.ankiweb.net
- 手写笔记

**自检清单**：
- [ ] 复习旧词汇
- [ ] 学习新词汇
- [ ] 大声朗读

---

#### 09:15-09:30 | 发音和听力 (15分钟)

**目标**：改进发音，熟悉英文语调

**具体任务**：
1. 选择一个1-2分钟的短视频
2. 第一遍：仔细听，不看字幕
3. 第二遍：看字幕，跟读
4. 第三遍：暂停，复述关键句子

**推荐资源**：
- Rachel's English: www.youtube.com/@rachels_english
- BBC Learning English: www.bbc.com/learningenglish
- Easy English: www.youtube.com/@easyenglish1

**推荐视频时长**：
- 初级：3-5分钟
- 中级：5-10分钟
- 高级：10-15分钟

**自检清单**：
- [ ] 不看字幕听一遍
- [ ] 看字幕跟读一遍
- [ ] 复述关键表达

---

#### 09:30-09:45 | 语法学习 (15分钟)

**目标**：系统学习英语语法

**第1-2周**：学习基础时态
- 一般现在时 (Simple Present)
- 一般过去时 (Simple Past)

**第3-4周**：学习进阶时态
- 一般将来时 (Simple Future)
- 现在进行时 (Present Continuous)

**具体任务**：
1. 阅读语法笔记（5分钟）
2. 完成练习题（7分钟）
3. 造2个例句（3分钟）

**推荐资源**：
- 本项目中的"基础时态.md"
- Khan Academy: www.khanacademy.org
- YouTube语法频道

**自检清单**：
- [ ] 理解语法规则
- [ ] 完成练习
- [ ] 能正确造句

---

#### 09:45-10:00 | 口语预热 (15分钟)

**目标**：开始练习口语，为真实对话做准备

**具体任务**：
1. 选择今天的话题（30秒）
2. 用英文思考内容（3分钟）
3. 大声说出来（5分钟）
4. 录音（3分钟）
5. 回放检查（3分钟）

**今天话题建议**：
- "What did I do today?"
- "My favorite food is..."
- "I like to... because..."

**推荐工具**：
- 手机语音记录器
- Google Recorder
- Voice Memos (iPhone)

**自检清单**：
- [ ] 选好话题
- [ ] 大声练习
- [ ] 录音检查
- [ ] 听出错误并改正

---

### 🌤️ 午间计划（推荐 12:00-1:00 PM 或 2:00-3:00 PM）

#### 时段1 | 阅读理解 (20分钟)

**目标**：提高阅读速度和理解能力

**难度递进**：
- 第1-2周：简单故事或儿童读物
- 第3-4周：改编版名著或简化新闻
- 第5-8周：真实新闻文章
- 第9-12周：原版书籍或学术文章

**具体任务**：
1. 选择阅读材料（1分钟）
2. 快速浏览标题和首段（1分钟）
3. 仔细阅读全文（12分钟）
4. 查不认识的单词（3分钟）
5. 写出主要意思（3分钟）

**推荐资源**：
- News in Slow English: newsinslow.com
- VOA Learning English: learningenglish.voanews.com
- Medium: www.medium.com
- Wattpad: www.wattpad.com

**阅读笔记模板**：
```
标题：_______
主要内容：
新单词：（写5-10个）
我的想法：
```

**自检清单**：
- [ ] 理解总体意思
- [ ] 找到关键信息
- [ ] 学会新单词

---

#### 时段2 | 听力训练 (20分钟)

**目标**：提高听力理解能力

**方法**：精听（Intensive Listening）

**具体任务**：
1. 选择音频（3-5分钟）
2. 第一遍听，不看字幕（3分钟）
3. 写下能听到的单词（3分钟）
4. 看字幕，再听一遍（3分钟）
5. 总结内容，查新单词（8分钟）

**推荐资源**：
- BBC Learning English Podcast
- VOA Learning English: learningenglish.voanews.com
- YouTube频道：Rachel's English, Easy English
- Spotify播客

**初级建议**：
- 视频：3-5分钟
- 内容：日常对话
- 难度：有字幕

**中级建议**：
- 视频：5-10分钟  
- 内容：新闻或教学
- 难度：可选字幕

**高级建议**：
- 视频：10-15分钟
- 内容：新闻、讲座、纪录片
- 难度：无字幕

**听力笔记**：
```
来源：_______
日期：_______
主题：_______

能听出的关键词：

新表达：

理解度：___%
```

**自检清单**：
- [ ] 第一遍听出关键词
- [ ] 第二遍理解整体内容
- [ ] 学到新表达

---

#### 时段3 | 写作练习 (20分钟)

**目标**：提高写作能力

**周期递进**：
- 第1-2周：日记（80-100字）
- 第3-4周：简单段落（150字）
- 第5-8周：邮件或短文（200-300字）
- 第9-12周：论文或文章（400+字）

**具体任务**：
1. 选择写作主题（2分钟）
2. 列提纲（3分钟）
3. 自由写作（10分钟）
4. 自我检查语法（3分钟）
5. 记录在学习日记（2分钟）

**今周写作话题**：
- "My Day" - 描述今天发生的事
- "My Family" - 介绍你的家人
- "My Favorite Food" - 描述你最喜欢的食物
- "My Hobbies" - 列出你的爱好

**写作检查清单**：
```
☐ 主题清晰
☐ 内容相关
☐ 时态一致
☐ 拼写正确
☐ 标点合理
☐ 句子结构清晰
```

**自检清单**：
- [ ] 完成写作任务
- [ ] 检查基本语法
- [ ] 记录新的表达方式

---

### 🌙 晚间计划（推荐 7:00-8:00 PM）

#### 20分钟 | 口语交流或跟读

**方案A：与语言伙伴交流（20分钟）**

**准备**：
1. 提前列出3-5个讨论话题
2. 准备关键词汇
3. 设定时间限制

**具体过程**：
1. 问候和寒暄（2分钟）
2. 讨论第一个话题（6分钟）
3. 讨论第二个话题（6分钟）
4. 讨论第三个话题（4分钟）
5. 道别（2分钟）

**推荐平台**：
- Tandem: www.tandem.net
- Italki: www.italki.com
- HelloTalk: www.hellotalk.com

**对话评估**：
```
对话流畅度：☆☆☆☆☆
单词使用：☆☆☆☆☆
语法正确：☆☆☆☆☆
发音清晰：☆☆☆☆☆
理解能力：☆☆☆☆☆

我犯的错误：

下次改进：
```

---

**方案B：跟读和模仿（20分钟）**

**目标**：改进发音和语调

**具体任务**：
1. 选择一个2-3分钟的视频（2分钟）
2. 听一遍，注意语调和重音（3分钟）
3. 逐句暂停，跟读（10分钟）
4. 整段跟读（3分钟）
5. 录音自己的版本（2分钟）

**推荐视频**：
- TED Talks 的简短演讲
- 电影精彩对白
- 歌曲歌词
- 演讲或演讲片段

**自检清单**：
- [ ] 能跟上语速
- [ ] 发音接近原版
- [ ] 语调自然

---

3. 记录收获和困难（3分钟）

## 🔄 一周完整计划

### 周一-周五（工作日）

```
早上 (15分)：词汇 → 听力 → 语法 → 口语
中午 (20分)：阅读
下午 (20分)：听力
晚上 (5分)：写作 或 语言交换
总计：60分钟
```

### 周六（加强日）

```
上午 (30分)：阅读 + 口语交流
下午 (20分)：听力 + 语言交换
晚上 (10分)：周总结
总计：60分钟
```

### 周日（复习日）

```
上午 (20分)：复习周内容
中午 (20分)：阅读喜欢的内容
下午 (20分)：听力总复习
晚上 (10分)：计划下周
总计：70分钟（可选加时）
```

## 💪 坚持技巧

### 1. 同一时间学习
建议每天同一时间学习，形成习惯。比如：
- 早上 7:00-8:00
- 或 中午 12:00-1:00
- 或 晚上 7:00-8:00

### 2. 设定提醒
- 手机日历提醒
- 手机定时器
- 闹钟

### 3. 跟踪进度
- 打印每日计划
- 记录学习时间
- 定期检查

### 4. 准备好资源
- 提前下载视频
- 准备笔记本
- 测试设备

### 5. 找学习伙伴
- 互相鼓励
- 分享进度
- 一起学习

## 🎯 灵活调整

**如果你有更多时间（2小时）**：
- 每个板块增加30分钟
- 或添加电影/电视剧观看
- 或增加语言交换时间

**如果你只有30分钟**：
- 每个板块减少一半
- 优先选择阅读或听力
- 可跳过其中一个板块

**周末调整**：
- 更多休闲学习
- 增加口语交流
- 看感兴趣的内容
