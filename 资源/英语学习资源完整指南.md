## 📅 学习计划建议

### 每日资源使用计划

**上午（20分钟）**
- BBC Learning English 或 Rachel's English 视频
- 学习新表达或语法

**中午（15分钟）**
- 阅读简短新闻（News in Slow English）
- 学习词汇

**下午（15分钟）**
- 听力练习（播客或视频）
- 或 Duolingo/Quizlet

**晚上（10分钟）**
- 写日记或邮件
- 或 语言交换（如果有伙伴）

## 💡 资源使用技巧

### 初级阶段（第1-3月）
1. 重点：BBC Learning English + YouTube (Rachel/Emma)
2. 补充：Duolingo + Quizlet
3. 阅读：News in Slow English
4. 听力：VOA Learning English

### 中级阶段（第4-6月）
1. 重点：Easy English + TED Talks
2. 补充：Tandem 语言交换
3. 阅读：BBC News + Medium
4. 听力：新闻播客

### 高级阶段（第7-12月）
1. 重点：真实内容（电影、纪录片）
2. 补充：Italki 一对一课程
3. 阅读：原版书籍
4. 听力：Podcast + 新闻
